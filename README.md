# 🌵 Laboratório Oracle SQL - Inventário de Cactos

Este repositório contém o desenvolvimento de um banco de dados para catalogar uma coleção real de cactos e suculentas, servindo como laboratório prático para estudos de **DBA (Database Administration)**.

## 🚀 Desafios de Ambiente Superados
Durante a configuração, realizei o troubleshooting de segurança no Windows 11 para permitir a execução de extensões do **VS Code** e estabeleci a conectividade com o **Oracle 21c XE**.

## 🛠️ Tecnologias Utilizadas
- **Banco de Dados:** Oracle Database 21c Express Edition.
- **Ferramentas:** VS Code com extensão Oracle SQL Developer.
- **Linguagem:** SQL (DML e DDL).

## 📊 Estrutura da Tabela
A tabela `MEUS_CACTOS` armazena o ID, o nome popular e a espécie/tipo da planta, garantindo a integridade dos dados através de Chaves Primárias.
